<h1>Progress-bar</h1>
<h2>Описание</h2>
<p>Блок с колесом прогресса с тремя состояниями: </p>
<ul>
  <li>
    Выбор значения прогресса (полный круг равен 100)
  </li>
  <li>
    Анимация вращения выбранного отрезка прогресса по часовой стрелке
  </li>
  <li>
    Скрытие блока прогресса
  </li>
</ul>
